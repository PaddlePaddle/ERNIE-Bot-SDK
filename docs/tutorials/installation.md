# 安装

## pip快速安装

执行如下命令，快速安装最新版本ERNIE Bot Agent（要求Python >= 3.8)。

```shell
pip install --upgrade erniebot-agent
```

## 源码安装

```shell
git clone https://github.com/PaddlePaddle/ERNIE-Bot-SDK.git

# 安装ErnieBot
cd ERNIE-Bot-SDK/erniebot
pip install .

# 安装ErnieBot-Agent
cd ../erniebot-agent
pip install .
```
